# 目录说明

这个目录用于系统学习 GitHub 与 Git 的代码管理能力。

## 推荐阅读顺序

1. Git 与 GitHub 的区别
2. 免费版与付费版
3. 仓库与文件管理
4. 分支、Commit、Pull Request 与合并
5. Issues、Projects 与协作
6. Actions、Pages、Packages 与 Codespaces
7. 权限、安全与仓库保护
8. Git 安装、配置与认证
9. Git 常用命令
10. 一人开发标准工作流
11. AI 与 GitHub 协作
12. 常见错误与恢复

新手需要快速查命令时，可以直接打开 `QUICK-COMMANDS.md`。
