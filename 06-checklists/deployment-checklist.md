# 部署上线检查清单

上线前与上线后确认：

## 上线前

- [ ] 目标分支和版本正确。
- [ ] 构建命令成功。
- [ ] 生产环境变量完整且未进入仓库。
- [ ] 数据库已备份。
- [ ] 迁移脚本已在测试环境验证。
- [ ] 域名、HTTPS、存储和外部服务已配置。
- [ ] 日志与基本监控可用。
- [ ] 回滚方式明确。

## 上线后

- [ ] 生产地址真实可访问。
- [ ] 首页和静态资源加载正常。
- [ ] 登录与权限正常。
- [ ] 核心业务流程真实通过。
- [ ] 数据读写正常。
- [ ] 外部 API、邮件、支付等真实流程已验证。
- [ ] 页面无明显控制台错误。
- [ ] 日志无持续严重错误。
- [ ] 敏感信息未泄露。
- [ ] 版本、部署时间和已知问题已记录。

只有生成部署说明而没有真实线上结果时，不得标记“已上线”。
